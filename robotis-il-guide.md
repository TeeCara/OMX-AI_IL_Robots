# 📘 모방학습 개요 (Imitation Learning)

---

## 📌 1. 모방학습이란?

모방학습(Imitation Learning)은  
**전문가(사람)의 행동 데이터를 보고 그대로 따라 배우는 머신러닝 방법**이다.

즉, 정답을 직접 찾는 것이 아니라  
👉 **"이 상황에서는 이렇게 행동하면 된다"** 를 학습하는 방식이다.

---

## 🎯 2. 왜 사용하는가?

| 이유 | 설명 |
|------|------|
| 데이터 기반 학습 | 사람이 직접 행동한 데이터를 사용 |
| 빠른 학습 | 강화학습보다 빠르게 학습 가능 |
| 구현 쉬움 | 보상 함수 설계가 필요 없음 |

---

## 🧠 3. 학습 과정

1. 사람이 직접 행동 수행 (데이터 수집)
2. 상태(State)와 행동(Action) 기록
3. 모델이 이를 학습
4. 새로운 상황에서 동일하게 행동

---

## ⚙️ 4. 주요 개념

| 개념 | 설명 |
|------|------|
| State (상태) | 현재 상황 (예: 카메라 이미지) |
| Action (행동) | 해야 할 행동 (예: 왼쪽 이동) |
| Expert | 정답을 알려주는 사람 |
| Policy | 상태 → 행동으로 바꾸는 함수 |

---

## 🔥 5. 대표 방법

### 1. Behavior Cloning
- 가장 기본적인 방법
- 지도학습처럼 동작
- 입력 → 상태, 출력 → 행동

👉 단점:  
잘못된 상황에 가면 복구가 어려움

---

### 2. DAgger (Dataset Aggregation)
- 잘못된 상황 데이터도 추가로 학습
- 반복적으로 데이터 보완

👉 Behavior Cloning의 단점 보완

---

## 🚗 6. 활용 분야

| 분야 | 예시 |
|------|------|
| 자율주행 | 운전자의 운전 방식 학습 |
| 로봇 제어 | 사람의 동작 따라하기 |
| 게임 AI | 플레이어 행동 학습 |

---

## ⚠️ 7. 한계점

- 데이터에 없는 상황에 약함  
- 전문가 데이터가 많이 필요  
- 오류가 누적될 수 있음 

---
