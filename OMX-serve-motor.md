# ⚙️ OMX 서버모터 (ROBOTIS OMX) 개요

---

## 📌 1. OMX란?

ROBOTIS OMX는  
👉 **초보자를 위한 Physical AI 로봇 매니퓰레이터**이다.

- 데이터 수집 및 AI 학습(모방학습, 강화학습) 가능  
- 로봇을 직접 조작하며 AI를 학습할 수 있는 플랫폼  

---

## 🎯 2. 주요 특징

### ⚙️ 설정 필요 없음
- 모터가 공장에서 미리 설정됨  
- 별도의 초기 설정 없이 바로 사용 가능  

### 🔌 캘리브레이션 불필요
- 모든 모터가 사전 보정됨  
- 사용자가 따로 조정할 필요 없음  

### 🛡️ 안전한 초기화
- 어떤 위치에서도 초기 자세로 안전하게 복귀  

### 🧭 360도 회전
- 베이스가 360도 회전 가능  
- 작업 범위 확장  

### 🪶 가벼운 무게
- 휴대 및 설치가 쉬움  

### ✨ 부드러운 조작
- 저마찰 구조로 쉽게 조작 가능  
- 초보자도 사용 가능  

### 🔋 간편한 전원
- 일부 모델은 USB-C만으로 사용 가능  

---

## 🤖 3. 기술적 특징

| 항목 | 설명 |
|------|------|
| 자유도 | 5-DOF (5축 움직임) |
| 모터 | DYNAMIXEL-X 기반 |
| 제어 | ROS 2 지원 |
| AI | 모방학습(IL), 강화학습(RL) 지원 |

---

## 🔄 4. AI 활용

- 데이터 수집 → AI 학습 → 로봇 제어  
- LeRobot, ROS 2 등과 연동 가능  
- 실제 로봇 기반 AI 실험 가능  

---

## 🧩 5. 구성 종류

| 모델 | 설명 |
|------|------|
| OMX-L | 조작용 컨트롤러 (Leader) |
| OMX-F | 로봇 팔 (Follower) |
| OMX-AI | 전체 세트 (Leader + Follower) |

👉 Leader + Follower 세트로 사용

---

## 💡 6. 내가 이해한 점

OMX는  
👉 **AI를 실제 로봇에 적용해서 실습할 수 있는 장비**이다.

특히  
👉 설정 없이 바로 사용할 수 있어서  
초보자도 쉽게 시작할 수 있는 것이 큰 장점이다.

---

## 📌 7. 한줄 정리

👉 **AI 학습과 로봇 제어를 쉽게 할 수 있는 입문용 로봇 플랫폼**
